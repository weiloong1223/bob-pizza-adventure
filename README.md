# 🍕 Bob's Pizza Adventure

> An Interactive Showcase of AI-Powered Software Development Lifecycle (SDLC)

## 🤖 What is This?

**Bob's Pizza Adventure** is a fun, interactive web experience that demonstrates how AI (specifically IBM Bob) can assist throughout the entire Software Development Lifecycle. Watch Bob build a complete pizza ordering application from planning to deployment, with interactive mini-games at each phase!

## ✨ Features

### 🎮 Interactive SDLC Journey
Experience all phases of software development:
- 🎯 **Planning**: Requirements gathering & feature selection
- 🎨 **Design**: UI/UX wireframes & component builder
- 💻 **Coding**: Live code generation & bug spotting
- 🧪 **Testing**: Automated tests & bug squashing
- 🚀 **Deployment**: CI/CD pipeline & rocket launch

### 🍕 Working Pizza App
Try the actual pizza ordering app that Bob built!

- ⏱️ Time saved: 20 hours
- 📝 Lines of code: 342
- 🐛 Bugs prevented: 12
- ✅ Tests written: 28

## 🚀 Quick Start

### Windows
```batch
git clone https://github.com/weiloong1223/bob-pizza-adventure.git
cd bob-pizza-adventure
setup.bat
```

### Mac/Linux
```bash
git clone https://github.com/weiloong1223/bob-pizza-adventure.git
cd bob-pizza-adventure
chmod +x setup.sh
./setup.sh
```

### Manual
Simply open `index.html` in your web browser!

## 📁 Project Structure

```

├── index.html              # Main interactive showcase
├── pizza-app.html          # Working pizza ordering app
├── css/
│   └── styles.css          # All styling and animations
├── js/
│   ├── main.js             # Core functionality
│   ├── phases.js           # Phase-specific interactions
│   └── animations.js       # Visual effects
```

## 🎨 Customization

### Change Colors
Edit `css/styles.css` to modify the color scheme.

### Add More Pizzas
Edit the `pizzas` array in `pizza-app.html`.

### Modify Phases
Update phase content in `js/main.js`.
