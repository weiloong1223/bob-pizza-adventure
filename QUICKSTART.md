# 🚀 Quick Start Guide

## Get Started in 30 Seconds!

### Windows Users

1. **Download the project**
   ```batch
   git clone https://github.com/weiloong1223/bob-pizza-adventure.git
   ```

2. **Navigate to the folder**
   ```batch
   cd bob-pizza-adventure
   ```

3. **Run the setup**
   ```batch
   setup.bat
   ```

That's it! Your browser will open automatically. 🎉

---

### Mac/Linux Users

1. **Download the project**
   ```bash
   git clone https://github.com/weiloong1223/bob-pizza-adventure.git
   ```

2. **Navigate to the folder**
   ```bash
   cd bob-pizza-adventure
   ```

3. **Make setup executable and run**
   ```bash
   chmod +x setup.sh
   ./setup.sh
   ```

Your browser will open automatically. 🎉

---

### No Git? No Problem!

1. **Download ZIP**
   - Click the green "Code" button on GitHub
   - Select "Download ZIP"
   - Extract the ZIP file

2. **Open the file**
   - Double-click `index.html`
   - Or right-click → Open with → Your Browser

Done! 🎉

## 🆘 Troubleshooting

### Browser doesn't open automatically?
- Manually open `index.html` in your browser
- Recommended browsers: Chrome, Firefox, Edge, Safari

### Animations not working?
- Make sure JavaScript is enabled
- Try a different browser
- Check browser console for errors (F12)

### Setup script won't run?
- **Windows**: Right-click `setup.bat` → Run as Administrator
- **Mac/Linux**: Make sure you ran `chmod +x setup.sh`

### Files missing?
- Re-download the project
- Make sure you extracted all files from the ZIP
